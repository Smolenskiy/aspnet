
Программа выполняет создание, просмотр, редактирование и удаление записей.

Запись представляет из себя связанные поля с датой, текстом и именем проекта.

Так же предусматривается создание, просмотр, редактирование и удаление проектов.

Доступ к списку проектов и управлению ими можно получить, нажав на кнопку Projects в навигационном меню сверху.

Так же предусматривается возможность фильтрации записей по дате и имени проекта.

Существует возможность сортировки записей по дате и имени проекта путём нажатия на соответствующее имя столбца таблицы.

В проекте реализована пагинация, которая создаёт дополнительные страницы при наличии более 5 записей.



Пагинация и сотртировка реализованы так же с помощью тэг-хелперов.

Их код содержится в папке tag-helpers, но не подключён в представлениях, дабы не перегружать проект.
